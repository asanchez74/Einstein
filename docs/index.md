# Bienvenue dans Einstein

Bienvenue dans la documentation officielle d'Einstein.

Cette documentation décrit l'architecture, les choix techniques et les différents projets hébergés sur la plateforme.

## Philosophie

- Documentation versionnée avec Git.
- Développement local sous VS Code.
- Exécution sur Synology.
- Infrastructure reproductible.
- Sécurité avant tout.

## Projet en cours

**Einstein Maison**

Le premier prototype est consacré à la supervision et à l'automatisation de la maison.
